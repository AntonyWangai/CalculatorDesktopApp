# CalculatorDesktopApp
A modern desktop calculator application built with C# and WinUI 3 (.NET).

This project is designed as a learning and practice application for WinUI 3, MVVM architecture, and desktop application development using C#.

Features
Basic arithmetic operations (Add, Subtract, Multiply, Divide)
MVVM architecture with clear separation of UI and logic
Calculation history tracking
Responsive WinUI 3 interface
Modern Windows-native UI using Mica backdrop

Tech Stack
C#
.NET
WinUI 3
MVVM Pattern
XAML


Purpose
This project was built to:

Learn WinUI 3 desktop development
Practice the MVVM design pattern
Understand ICommand, data binding, and UI state management
Build a real-world styled Windows calculator application
Getting Started
Clone the repository
Open the project in Visual Studio 2022 or later
Restore NuGet packages
Run the application
